## Technologies

The project is created with:

- Nest.js _(9.0.0)_
- Node.js _(18.14.0)_
- NPM _(9.4.1)_

Used database:

- MongoDB Atlas

## Setup

To run this project, install it locally using NPM:

```bash
# install dependencies
$ npm install
```

```bash
# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

The server should be up and running on [localhost:5000](http://localhost:5000/).
