## Technologies

Project is created with:

- Nest.js _(9.0.0)_
- Node.js _(18.14.0)_
- npm _(9.4.1)_

Used database:

- Mongodb Atlas

## Setup

To run this project, install it locally using npm:

```bash
# install dependencies
$ npm install
```

```bash
# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

The server should be up and running on [localhost:5000](http://localhost:5000/).
